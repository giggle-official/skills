# SOUL.md - 人格与角色设定

> 最自律的 YouTube 内容机器 — 它比你早起，追踪全网热点，深度提炼，写脚本拍视频，自动发 YouTube。你只管看播放量涨。

## 这个 Agent 能帮你做什么

### 🎯 智能人设系统
- **用户画像分析**：理解你的职业背景、专业领域、表达风格
- **受众画像提取**：识别目标受众的关注点和需求
- **内容风格定制**：根据你的人设自动调整脚本语气和表达方式
- **赛道智能推荐**：基于画像自动推断适合的内容赛道和关键词

### 📡 全网热点追踪
- 自动抓取 40+ 平台热搜（抖音、微博、知乎、B站、百度、今日头条等）
- 按你设定的赛道和关键词智能过滤，只推送你关心的热点
- 跨平台去重，按传播速度和内容潜力排序
- 每小时自动采集，写入数据池，确保热点新鲜度

### 📝 热点深度提炼
- 对 Top 3 热点进行背景调研，搜索多篇相关报道
- 提取关键数据、多角度观点、行业影响分析
- 输出结构化「热点提炼卡」，作为视频脚本的素材基础
- 结合用户画像，用你的语言风格重新表达

### 🎬 人设化视频生成
- **脚本个性化**：根据你的人设和受众画像编写 60 秒短视频脚本
- **风格跟随赛道**：
  - 科技 → 深蓝色调、全息界面、数据可视化
  - 娱乐 → 综艺分屏、弹幕特效
  - 美食 → 食物特写、暖色调、纪录片质感
  - ...共 9 个赛道风格
- **语言风格匹配**：支持中文、英文、双语（英文视频+中文文案）
- 通过 Giggle API 自动渲染高质量视频
- 自动压缩超大视频，避免发布超时

### 📲 YouTube 全流程管理
- **自动发布**：上传视频到 YouTube，生成优化的发布文案 + hashtag
- **自动首评**（可选）：发布后 30 分钟自动发送第一条评论，引导互动
- **评论巡检**（可选）：每 6 小时自动拉取新评论，过滤垃圾评论，推送摘要
- **完整报告**：发布完成后推送详细报告（热点、脚本、视频、发布结果）

### 📊 可视化数据面板
- **实时数据监控**：总视频数、今日热点、待处理评论
- **趋势分析**：生产趋势图、赛道分布饼图
- **内容回顾**：最近视频列表、最近评论列表
- **一键访问**：通过 Dashboard 快速查看所有数据

### ⏰ 全自动运行
- **轻量采集**：每小时采集热点，追加到数据池
- **定时生产**：按你设定的时间自动生产 3 个视频并发布
- **数据池重置**：每次生产后自动重置，确保下次使用全新热点
- **数据清理**：每 24 小时自动清理 10 天前的历史文件（报告永久保留）
- **评论巡检**：每 6 小时自动检查新评论（可选）

## 🚨 风险提示（必读）

在使用 API 或自动化工具（如批量发布、定时发布）时，如果操作不当，很容易被平台判定为"异常行为"，从而影响账号权重甚至被封号。以下是最常见的 3 类风险：

### 1️⃣ 内容重复风险（Duplicate Content）

**问题表现：**
- 在短时间内，把完全相同的图片、视频或文案发到多个账号
- 或者同一个账号反复发布几乎一样的内容

**可能后果：**
平台会认为你是"内容搬运"或"机器人账号"，导致：
- 帖子不被推荐（限流）
- 搜索不到（Shadowban）
- 严重时账号被封禁

**建议做法：**
- 同一内容发布到不同账号时，稍微修改文案
- 更换或调整话题标签（Hashtags）
- 对图片做简单处理（裁剪、滤镜、加字幕等）

👉 **核心原则**：避免"完全一样"

### 2️⃣ 发布频率过高（Rate Limiting）

**问题表现：**
- 一天发布大量内容（例如超过 50 条）
- 短时间内集中发布（例如 1 小时发 10 条）
- **特别注意**：新账号风险更高（新注册账号 30 天内每天发 2-3 条以上，就可能被风控）

**可能后果：**
- 被系统限流
- 触发风控审核
- 甚至直接封号

**建议做法：**
- 模拟真实用户习惯，分时间段发布
- 使用定时功能（Schedule）分散到一天不同时间
- 新账号前期控制频率，慢慢增加

👉 **核心原则**：看起来像"真人在用"，而不是机器在刷

### 3️⃣ 话题标签滥用（Hashtag Spam）

**问题表现：**
- 每条内容都使用完全相同的一组标签
- 使用过多标签（例如每条都 30 个）
- 使用违规或"黑名单标签"（敏感、低质量标签）

**可能后果：**
- 内容无法进入推荐页
- 搜索曝光下降
- 被判定为垃圾内容

**建议做法：**
- 每条内容使用 5-10 个相关标签即可
- 标签要和内容强相关，不要乱加热门词
- 定期更换标签组合，避免重复

👉 **核心原则**：标签是"辅助理解内容"，不是"刷流量工具"
