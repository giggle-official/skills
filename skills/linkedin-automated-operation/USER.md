# USER.md - 用户偏好与设置

## 首次设置（激活前必需）

当你第一次使用这个 agent 时，它会引导你完成一次性配置向导。你必须提供：

### 🔑 API 与凭证配置
- **Giggle API Key**（必需）— 用于通过 `giggle-generation-image` 技能生成高质量图片
  - 获取方式：[giggle.pro](https://giggle.pro) → 账号设置 → API Key
- **X2C API Key**（必需）— 用于 LinkedIn 帖子发布
  - 获取方式：访问 [x2creel.ai](https://www.x2creel.ai) → 注册账号 → 关联 LinkedIn → Developer API Key

### 🎯 内容方向定义（关键步骤）

告诉 agent 追踪什么样的热点、生成什么样的内容：

| 设定项 | 说明 | 示例 |
|--------|------|------|
| **赛道/领域** | 内容垂直方向，可选多个 | 科技数码、财经理财、娱乐八卦、美食探店、健身运动、教育知识、汽车、时尚穿搭、游戏电竞、职场成长 |
| **关注关键词** | 优先匹配包含这些词的热点 | AI、iPhone、比特币、减肥、考研 |
| **排除关键词** | 包含这些词的热点自动过滤 | 政治、军事、灾难、血腥 |
| **目标受众** | 你的观众画像 | Z世代、职场白领、宝妈、大学生 |

### 🎭 发布者画像（个性化文案的关键）

让 agent 知道"你是谁"，这样同一条热点新闻会从你的角度切入：

| 设定项 | 说明 | 示例 |
|--------|------|------|
| **角色定位** | 你在 LinkedIn 上的身份 | AI startup CEO、高级工程师、产品设计师、在校研究生 |
| **语气风格** | 帖子的说话方式 | professional but opinionated, data-driven |
| **分析视角** | 看热点时习惯从什么方向切入 | 商业价值和市场机会、技术架构和工程实践 |
| **历史风格** | 自动从你的 LinkedIn 最近 20 条帖子中分析 | 开场方式、句式偏好、emoji 习惯等（需 ≥ 5 条历史帖子） |

> 💡 角色、语气、视角由你手动填写；历史风格由 agent 自动分析补充。如果历史帖子不足 5 条，会跳过自动分析，仅依赖你的手动画像。

### 🖼️ 图片风格设置

根据赛道自动推荐风格，也可自选：

| 赛道 | 风格 | 画面感 |
|------|------|--------|

| 美食探店 | 美食纪录片 | 食物特写+暖色调 |

## 运行方式

- **自动运行**（配置完成后立即生效）：
  - 每 1 小时：静默采集全网热点，写入数据池
  - 按用户配置的时间：从数据池取 Top 3，生成 3 张图片，间隔 10 分钟依次发布到 LinkedIn，完成后推送报告

- **手动触发**：随时说 `run now` 立即执行完整流程

## 交互风格

- 每个阶段完成后发送简短状态更新
- 流水线完成后推送完整内容产出报告
- 错误立刻告知，附原因和建议

## 边界与范围

### ✅ 这个 Agent 做什么
- 每小时从 40+ 平台收集热点，写入数据池
- 按用户配置的时间取 Top 3，为每条热点进行深度调研和提炼
- 为每条热点生成一张行业风格化图片
- 间隔 10 分钟依次将 3 张图片发布到 LinkedIn
- 发布完成后推送完整报告（含提炼卡 + 发布文案 + 图片信息）

### ❌ 这个 Agent 不做什么
- **不发布到 LinkedIn 以外的平台**（专注 LinkedIn）
- 不访问你的平台私信或个人数据
- 不生成违反平台社区准则的内容
- 不在配置文件之外存储凭证
