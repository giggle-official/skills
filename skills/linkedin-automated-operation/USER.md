# USER.md - 用户偏好与设置

## 首次设置（激活前必需）

首次使用时，Agent 会引导你完成一次性配置。你需要提供：

### API 与凭证
- **Giggle API Key**：用于 AI 图片生成（`giggle-generation-image`）
- **X2C API Key**：用于 LinkedIn 图文发布（`x2c-socialposter`）

### 内容方向
- 赛道/领域（可多选）
- 关注关键词
- 排除关键词
- 目标受众画像

### 图片生成偏好
- 图片风格（如专业资讯卡、数据海报、极简插画）
- 图片模型（默认 `seedream45`）
- 画幅比例（推荐 `1:1` 或 `4:3`）
- 分辨率（推荐 `2K`）

---

## 运行方式

- **自动运行**
  - 每 1 小时：静默采集热点并写入数据池
  - 按配置时间：从数据池取 Top 3，生成 3 组「LinkedIn 文案 + 配图」，依次发布

- **手动运行**
  - 输入 `run now` 立即触发完整流程

---

## Agent 做什么

- 热点采集、筛选、提炼
- 生成 LinkedIn 风格文案
- 调用 `giggle-generation-image` 生成配图
- 发布图文到 LinkedIn
- 产出报告与日志

## Agent 不做什么

- 不生成或发布视频
- 不发布到 LinkedIn 以外的平台
- 不在配置文件外存储凭证
