# USER.md - 使用指南

## 快速指令

| 指令 | 效果 |
|------|------|
| **一键制作** | 采集热点 → 筛选 → 生成视频 → 自动发布 |
| **查看热点** | 获取全网实时热搜趋势 |
| **做一个关于 [话题] 的视频** | 指定话题生成视频 |
| **查看今日任务** | 所有 Task 状态一览 |
| **查看 TASK-xxx 详情** | 单个任务详情 |
| **查看收益** | 账户余额 + 收益流水 |
| **查看余额** | 积分 / X2C / USDC |
| **修改配置** | 调整赛道、关键词、视频参数 |
| **绑定账号** | 绑定 X2C API Key |
| **每日报告** | 订阅/取消每日运营数据邮件 |
| **draft mode** | 只制作不发布，完成后等你确认 |

你也可以直接发数字 1-9 触发对应功能。

## 一键制作流程

```
采集 → 筛选 → 制作(10-20分钟) → 发布 → 审核
```

- 每次制作消耗约 309 积分（剧本10 + 视频299，按60秒计）
- 视频完成后自动发布，无需手动确认
- 发布后由 X2C 平台审核，通过后自动获得流量分成资格
- 全程自动，你只需等待通知

## 收益说明

- 收益是账户级别的，按每日释放，不按单个视频计算
- 说 **查看收益** 可以看到积分、X2C 代币、USDC 余额和流水

## 配置说明

你的内容方向和视频参数保存在 config.json 中，随时可以修改：

- **赛道 / 关键词** — 决定筛选哪些热点
- **视频风格 / 分类 / 时长** — 决定视频怎么做
- **自动发布开关** — 想手动控制发布可以关掉

## 注意事项

- 首次使用前必须绑定 X2C 账号（去 https://www.x2creel.ai 获取 API Key）
- 视频制作需要 10-20 分钟，期间可以做其他事
- 制作失败不会自动重试（每次都扣积分），会报告原因让你决定
- 15 天内不会重复制作相同话题

## 每日报告

订阅后，每天会收到一封运营数据邮件，包含：
- 总收入、今日收益、播放量、ROI
- 7日收益趋势、平台分布
- 正在赚钱的作品 Top 10

可以随时开启/关闭，或调整接收时间（默认北京时间 09:00）。
