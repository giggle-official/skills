# USER.md - 用户偏好与设置

## 首次设置（激活前必需）

当你第一次使用这个 agent 时，它会引导你完成一次性配置向导。你必须提供：

### 🔑 API 与凭证配置
- **Giggle API Key**（必需）— 用于通过 `giggle-generation-drama` 技能生成高质量短剧视频
  - 获取方式：[giggle.pro](https://giggle.pro) → 账号设置 → API Key
- **X2C API Key**（必需）— 用于 Instagram 视频发布
  - 获取方式：访问 [x2creel.ai](https://www.x2creel.ai) → 注册账号 → 关联 Instagram → Developer API Key

### 🎯 内容方向定义（关键步骤）

告诉 agent 追踪什么样的热点、生成什么样的内容：

| 设定项 | 说明 | 示例 |
|--------|------|------|
| **赛道/领域** | 内容垂直方向，可选多个 | 科技数码、财经理财、生活方式、美食探店、健身运动、教育知识、旅行摄影、时尚穿搭、艺术创意、职场成长 |
| **关注关键词** | 优先匹配包含这些词的热点 | AI、iPhone、比特币、极简生活、数字游民 |
| **排除关键词** | 包含这些词的热点自动过滤 | 政治、军事、灾难、血腥 |
| **目标受众** | 你的观众画像 | Z世代、职场白领、创业者、设计师、摄影师 |

## 运行方式

- **自动运行**（配置完成后立即生效）：
  - 每 1 小时：静默采集全网热点，写入数据池
  - 按用户配置的时间：从数据池取 Top 3，生成 3 个视频，间隔 15 分钟依次发布到 Instagram，完成后推送报告

- **手动触发**：随时说 `run now` 立即执行完整流程

## 交互风格

- 每个阶段完成后发送简短状态更新
- 流水线完成后推送完整内容产出报告
- 错误立刻告知，附原因和建议

## 边界与范围

### ✅ 这个 Agent 做什么
- 每小时从 40+ 平台收集热点，写入数据池
- 按用户配置的时间取 Top 3，为每条热点进行深度调研和提炼
- 为每条热点生成一个 60 秒 Instagram 美学风格化视频
- 间隔 15 分钟依次将 3 个视频发布到 Instagram Reels
- 发布完成后推送完整报告（含提炼卡 + 发布文案 + 视频信息）

### ❌ 这个 Agent 不做什么
- **不发布到 Instagram 以外的平台**（专注 Instagram）
- 不访问你的平台私信或个人数据
- 不生成违反平台社区准则的内容
- 不在配置文件之外存储凭证
