# Product Poster Prompt Templates

Three templates adapted from the BASE framework for product poster generation.

---

## 1. E-commerce Product Launch 

**Use when**: Taobao/JD/Xiaohongshu product main image, new product launch

**Aspect ratio**: 3:4 or 1:1
**Model**: nano-banana-2

**Prompt template**:
```
E-commerce product poster for [PRODUCT NAME]. [KEY SELLING POINT].
Style: [STYLE — e.g. clean minimalist / bold vibrant], [COLOR PALETTE — e.g. pure white background with gold accents], commercial advertising aesthetic.
Visual: Keep the provided product as the hero element, centered composition, enhance with studio lighting and sharp product details. Add [COMPLEMENTARY ELEMENTS — e.g. scattered ingredient particles / soft bokeh backdrop / subtle shadow].
Text layout: Bold main title "[PRODUCT NAME]" at top, tagline "[SLOGAN]" below, price badge "[PRICE]" if applicable. Clean sans-serif typography.
Technical: High-end product photography quality, 2K resolution, no watermark, white or gradient background, commercial grade.
```

**Example**:
```
E-commerce product poster for AuraGlow Facial Serum. 30-second visible brightening with niacinamide complex.
Style: Luxury minimalist, pearl white background with soft rose gold accents, premium skincare aesthetic.
Visual: Keep the provided serum bottle as the hero element, centered composition, enhance with soft studio lighting and subtle bokeh. Add scattered rose petals and gold dust particles around the product.
Text layout: Bold main title "AuraGlow" at top, tagline "Glow in 30 Seconds" below product, clean serif typography.
Technical: High-end product photography quality, 2K resolution, no watermark, pearl white gradient background, commercial grade.
```

## 2. Brand Event / Campaign 

**Use when**: Launch events, seasonal promotions, anniversary campaigns

**Aspect ratio**: 9:16 or 16:9
**Model**: nano-banana-2

**Prompt template**:
```
Brand event promotional poster for [BRAND NAME] [EVENT NAME]. [EVENT DATE/THEME].
Style: [STYLE — e.g. grand luxurious / energetic modern], [COLOR PALETTE], [MOOD — e.g. celebratory / exclusive / exciting].
Visual: Feature the provided product prominently, integrate it into [SCENE/IMAGERY — e.g. dramatic stage / abstract geometric backdrop], dynamic composition with strong visual hierarchy.
Text layout: Event name "[EVENT NAME]" as dominant headline, date "[DATE]" prominent, venue "[VENUE]", brand logo placement at bottom center.
Technical: Cinematic quality, high contrast, sharp typography, no watermark, print-ready resolution.
```

**Example**:
```
Brand event promotional poster for NexTech 2025 Annual Launch Conference. Theme: Beyond Boundaries.
Style: Futuristic tech aesthetic, deep navy and electric blue with neon cyan accents, mood: visionary and powerful.
Visual: Feature the provided product as the focal point, integrate it into an abstract digital universe with glowing data streams, subtle hexagonal grid overlay, dramatic depth of field.
Text layout: "NexTech 2025" as dominant headline in bold tech font, "Beyond Boundaries" as tagline, "March 15 · Shanghai Exhibition Center" in clean sans-serif, logo bottom center.
Technical: Cinematic quality, high contrast, sharp typography, no watermark, print-ready resolution.
```

## 3. Social Media Promotion 

**Use when**: WeChat Moments, Xiaohongshu, Instagram posts

**Aspect ratio**: 9:16 (vertical) or 1:1 (square)
**Model**: nano-banana-2-fast

**Prompt template**:
```
Social media promotional image for [BRAND/PRODUCT]. [CAMPAIGN MESSAGE].
Style: [STYLE — e.g. trendy Gen-Z / warm lifestyle / bold streetwear], [COLOR PALETTE], scroll-stopping visual impact.
Visual: Keep the provided product as the centerpiece, place it in [LIFESTYLE SCENE — e.g. outdoor cafe / kitchen counter / gym setting], with [MOOD ELEMENTS — e.g. natural golden hour light / vibrant street background], authentic and relatable composition.
Text overlay: Short punchy headline "[HEADLINE]", subtle brand handle "@[HANDLE]", minimal text to maximize visual impact.
Technical: Mobile-optimized vertical format, vibrant colors, high engagement aesthetic, no watermark.
```

**Example**:
```
Social media promotional image for BrewCraft Cold Brew Coffee. Summer refreshment campaign.
Style: Trendy lifestyle aesthetic, warm amber and cool teal contrast, energetic summer vibes, scroll-stopping visual impact.
Visual: Keep the provided coffee bottle/cup as the centerpiece, place it on an outdoor cafe terrace with natural golden hour sunlight, condensation and ice cubes as props, lifestyle photography feel.
Text overlay: "Your Summer Essential" as punchy headline, "@BrewCraft" handle bottom right, minimal text to maximize visual impact.
Technical: Mobile-optimized vertical format, vibrant saturated colors, high engagement aesthetic, no watermark.
```

## Style Reference Guide

| Style Keyword | Description | Best For |
|--------------|-------------|----------|
| `clean minimalist` | White space, single hero, simple palette | Skincare, tech, premium |
| `bold vibrant` | High contrast, saturated colors, strong type | Food, fashion, youth |
| `luxury premium` | Gold/silver accents, dark bg, elegant type | Jewelry, spirits, luxury goods |
| `tech futuristic` | Dark bg, neon accents, geometric elements | Electronics, SaaS, gaming |
| `warm lifestyle` | Natural light, earthy tones, human element | Food, wellness, home goods |
| `playful youthful` | Bright colors, fun shapes, casual type | Snacks, toys, beauty |
